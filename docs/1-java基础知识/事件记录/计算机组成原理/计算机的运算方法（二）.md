# 计算机的运算方法

**在计算机中，小数点不用专门的器件表示，而是按约定那个的方式标出，共有两种方法表示小数的存在，即定点表示和浮点表示。**

*定点表示的数称为定点数，浮点表示的数称·为浮点数。*

## 定点表示

小数点固定在某一位置的数为定点数，

**表示方法：**

- 当小数点位于数符和第一数值位之间时，机器内的数为纯小数z 当小数点位于数值位之后时，机器内的数为纯整数。

- 采用定点数的机器称为定点机。

- 在定点机中，由于小数点的位置固定不变，故当机器处理的数不是纯小数或纯整数时，必须乘上一个比例因子，否则会产生“溢出”。
